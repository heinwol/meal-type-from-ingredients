"Foodie Favorites" uses Natural Language Processing (NLP) techniques to classify food mentions in Yelp reviews and subsequently matches those mentions to menu items to help users find out what dishes are most popular. This project was undertaken as my Capstone during the Galvanize Data Science Immersive program. For a step-by-step account of the project process, please see my project github page. Please note that the restaurant data shown above is from an older Yelp Academic Dataset, which contains reviews from 2006-2012, so both restaurant and menu data may have changed since then. The link to this dataset is on my github.
